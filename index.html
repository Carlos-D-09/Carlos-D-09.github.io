El servidor desarrollado en este proyecto esta pensado para llevar a cabo tres tareas:

  • Identificación de objetos en imágenes
  • API's de consumo para la interacción con una aplicación móvil
  • Implementación de una página web para la administración de la aplicación móvil

Para la identificación de objetos en imágenes, se entrenó una red neuronal mediante Python y la librería de TensorFlow, usando como base una red neuronal convolucional pre-entrenada (Inception V3).

Para el desarrollo de las API's y el back-end de la página web, se utilizo Flask. Para el desarrollo del front-end se utilizó HTML5 y CSS3 sin ninguno Framework CSS de por medio. Para hacer interactivo el diseño se hizo uso de JavaScript con la librería jQuery.
